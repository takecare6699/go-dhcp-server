# DHCP Server 测试快速开始指南

## 🚀 快速开始

### 1. 运行所有测试

```bash
# 使用测试脚本（推荐）
./run_tests.sh

# 或者直接使用 go test
go test -v dhcp_comprehensive_test.go
```

### 2. 查看测试结果

```bash
# 查看测试报告
cat test_reports/test_report_*.txt

# 查看详细日志
cat test_reports/detailed_test_output_*.log
```

## 🐛 常见问题

### 问题1: 权限不足
```bash
# 解决方案
sudo chmod +x run_tests.sh
```

### 问题2: 端口被占用
```bash
# 检查端口占用
lsof -i :8080

# 停止占用进程
kill -9 <PID>
```

### 问题3: 依赖缺失
```bash
# 安装依赖
go mod tidy
```
